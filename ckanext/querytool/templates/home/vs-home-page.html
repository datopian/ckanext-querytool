{{ _('How to use the Platform') }}

{% trans%}Four main steps are necessary to successfully design an application, with the first one taking place outside the Platform.{% endtrans %}

  • {{_('Step 1 - Prepare data')}}
    {% trans %}Before uploading the data into the Platform, it needs to be cleaned, validated and, depending on its source, transformed into a tabular dataset.{% endtrans %}
  • {{_('Step 2 – Load datasets')}}
    {% trans %}In the Load datasets tab, the prepared data is uploaded through familiar menus and metadata is filled to help identify and differentiate the datasets.{% endtrans %}
  • {{_('Step 3 – Design application')}}
    {% trans %}In the Design application tab, drop-down menus are used to add data-driven visualizations (charts, maps and tables), support elements (text and images) and interactive elements (filters, download options, sharing options). These elements can be combined and arranged in different ways to provide a user-friendly experience for exploring public health data.{% endtrans %}
